Question-1: What are the different ways to select an element in the DOM?
In JavaScript, you can select DOM elements using methods like getElementById (for a unique ID), getElementsByClassName and getElementsByTagName (for multiple elements by class or tag), or querySelector and querySelectorAll (for flexible CSS-based selection).
Question-2: What is the difference between innerHTML, innerText, and textContent?
innerHTML gets or sets the HTML content inside an element, interpreting tags as actual HTML, while innerText retrieves or sets only the visible text, ignoring hidden elements and extra spaces. textContent is similar to innerText but includes hidden text and treats everything as plain text without parsing HTML.
Question-3: What is event delegation in the DOM?
Event delegation lets you attach a single event listener to a parent element instead of adding listeners to multiple child elements, which is useful for dynamically added elements.
Question-4: What is event bubbling in the DOM?
Event bubbling happens when an event triggered on a child element propagates up through its parent elements, allowing them to handle the event too.
Question-5: How do you create, add, and remove elements using JavaScript?
To create an element, use document.createElement("tag"), to add it, use appendChild() or append(), and to remove it, use removeChild() or remove().
